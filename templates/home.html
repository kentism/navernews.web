주요 기능

키워드 검색

다중 키워드 자동 완성·최근 검색어 저장

원클릭 클리핑

기사 제목·링크·출처·날짜를 지정 포맷으로 바로 저장

다크 모드

눈에 편안한 다크 테마 자동 전환

클리핑 공유

복사·다운로드·클립보드에 바로 붙여넣기

간단 3단계 사용법

  1. 키워드 입력 → 검색

    검색창에 원하는 키워드를 입력하고 엔터 또는 검색 버튼을 누릅니다.

  2. 관심 기사 → 클립

    검색 결과 카드 오른쪽의 📌 버튼을 클릭하면 클리핑 텍스트 영역에 자동 추가됩니다.

  3. 클리핑 관리

    ‘클리핑’ 탭에서 복사·전체 삭제·다시 불러오기를 할 수 있습니다.
